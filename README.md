# WebGym - AI Motion Analysis Game

WebGym is an interactive web application that uses your webcam and AI motion detection (MediaPipe) to track your exercises.

## Features
- **Real-time Pose Detection**: Uses MediaPipe Pose to track body movements.
- **Motion Scoring**: Calculates scores based on your movements (e.g., squats, jumps).
- **YouTube Integration**: Play workout videos and control playback with your movements.
- **Responsive Design**: Works on both PC and mobile devices.

## How to Use
1. Allow webcam access when prompted.
2. Stand back so your full body (or at least upper body) is visible.
3. Follow the movements in the video!

## Technologies
- HTML5, CSS3, JavaScript
- MediaPipe Tasks Vision
- YouTube IFrame Player API
